Features

WSyncPro is designed to provide robust file management, synchronization, and archiving capabilities. Below are the key features that make WSyncPro a versatile and reliable tool.

Key Features

1. File Management

  • Copy, move, and clean files based on user-defined criteria.
  • Robust error handling and logging for file operations.
  • Integration with versioning to maintain historical snapshots of files.

2. Synchronization

  • One-way and two-way synchronization between directories or devices.
  • Conflict detection and resolution.
  • Scheduling options for automated synchronization.

3. Archiving

  • Compress files into zip archives with customizable grouping patterns.
  • Include metadata to improve file retrieval and organization.

4. State Management

  • Persistent state management for ongoing operations.
  • Caching mechanisms to improve performance and reduce database overhead.

5. External Device Detection

  • Detect and interact with external USB devices seamlessly.
  • Execute jobs tailored to specific devices as they are connected.

6. Data Import

  • Efficiently import files with advanced filtering options.
  • Automatically generate destination paths based on rules and templates.

7. User-Friendly Interface

  • Intuitive UI with a dark mode theme.
  • Live logs and job status tracking.
  • Tab-based navigation for managing imports, sync jobs, and settings.

8. Cross-Platform Support

  • Fully compatible with Windows and Linux systems.
  • Platform-specific implementations to optimize performance.

9. Logging and Reporting

  • Comprehensive logging for debugging and monitoring.
  • Detailed reports for each job, including success and failure rates.

Planned Features

1. Re-Rendering Services

  • Re-render video files using HandBrakeCLI for format conversion and optimization.
  • Support for batch processing of video files.

2. AutoSync

  • Automatically detect changes in source directories and trigger synchronization.
  • Provide real-time updates and conflict resolution.
